```python
def solution(absolutes, signs):
    
    return sum([absolutes[x] if signs[x]==True else -1*absolutes[x] for x in range(len(signs))])
```


```python
absolutes =[4,7,12]
signs = [True,False,True]
```


```python
solution(absolutes, signs)
```
