```python
n = 45
```


```python
def solution(n):

    max_pos = 1
    while n/(3**max_pos) >= 1:
        max_pos+=1

    result = []
    for i in range(max_pos)[::-1]:
        for j in range(3)[::-1]:
            if n-(3**i*j) >= 0:
                n -= (3**i*j)
                result.append(str(j))
                break

    return int(''.join(result)[::-1],3)
```


```python
solution(n)
```
