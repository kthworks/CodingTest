```python
def solution(lottos, win_nums):
    
    windict = {6:1, 5:2, 4:3, 3:4, 2:5, 1:6, 0:6}
    lowest = len(set(lottos).intersection(set(win_nums)))
    highest = lowest + len([x for x in lottos if x==0])
    
    answer = [windict[highest], windict[lowest]]
    return answer
```
